# Demo 功能说明

## 已恢复的功能

### 1. 材料系统
- **17种材料**: Sand, Water, Rock, Fire, Lava, Wood, Oil, Gunpowder, Acid, Ice, Steam, Glass, Gold, Diamond, Poison, Blood, Slime
- **快捷键选择**: 
  - 数字键 1-0: 前10种材料
  - 字母键 Q-U: 后7种材料

### 2. 画笔工具
- **可调大小**: 1-50像素
- **左键绘制**: 绘制当前选择的材料
- **右键擦除**: 擦除像素
- **圆形画笔**: 使用距离判断,绘制圆形区域

### 3. 相机系统
- **平移**: 方向键或WASD
- **缩放**: 鼠标滚轮 (0.1x - 10x)
- **边界限制**: 相机不会超出世界范围
- **窗口调整**: 支持窗口大小调整

### 4. 调试功能
- **F1**: 切换调试模式
- **F2**: 显示/隐藏块边界
- **F3**: 显示/隐藏FPS
- **块边界可视化**: 绿色半透明网格

### 5. 世界生成
- **G键**: 生成洞穴地形
- **H键**: 生成地表地形
- **C键**: 清空世界

### 6. 存档系统
- **Ctrl+S**: 保存世界到 world_save.bin
- **Ctrl+L**: 从 world_save.bin 加载世界

### 7. 暂停功能
- **空格键**: 暂停/恢复模拟

### 8. 性能统计
- **FPS**: 每秒帧数
- **Update Time**: 更新耗时(毫秒)
- **Render Time**: 渲染耗时(毫秒)
- **实时显示**: 控制台和屏幕

### 9. 渲染优化
- **颜色变体**: 每个像素有随机颜色变化
- **温度效果**: 高温像素显示红色
- **发光效果**: 发光材料更亮
- **纹理流式更新**: 高效渲染

## 示例场景

### 火山场景
1. 按5选择Lava
2. 在底部绘制岩浆池
3. 按4选择Fire
4. 在岩浆上方绘制火焰
5. 观察火焰和岩浆的交互

### 水坝场景
1. 按3选择Rock
2. 绘制一个水坝
3. 按2选择Water
4. 在水坝上游倒水
5. 观察水的流动和堆积

### 爆炸场景
1. 按8选择Gunpowder
2. 绘制一堆火药
3. 按4选择Fire
4. 在火药上点火
5. 观察爆炸效果

### 腐蚀场景
1. 按3选择Rock
2. 绘制一些岩石
3. 按9选择Acid
4. 在岩石上倒酸
5. 观察岩石被腐蚀

## 技术细节

### 更新系统
- **棋盘格更新**: 避免相邻块同时更新
- **从下到上**: 模拟重力
- **随机方向**: 避免偏向
- **两遍更新**: 奇偶块交替

### 渲染系统
- **纹理流**: SDL_TEXTUREACCESS_STREAMING
- **像素格式**: RGBA8888
- **颜色计算**: 实时计算变体和效果
- **相机裁剪**: 只渲染可见区域

### 材料系统
- **JSON配置**: config/materials/basic_materials.json
- **运行时加载**: MaterialRegistry单例
- **ID映射**: 名称到ID的映射
- **属性验证**: 加载时验证材料属性
