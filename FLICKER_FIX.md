# 画面闪烁问题修复总结

## 📋 问题分析

### 问题描述

**画面在闪烁**,屏幕不断闪烁,影响视觉体验。

### 问题根源

**`pass_callback_`在每个pass后都会被调用,这会导致渲染4次,造成闪烁!**

**问题代码**:
```cpp
void World::update(float dt) {
    // ...
    
    // 棋盘格4-pass更新
    for (int pass = 0; pass < 4; ++pass) {
        // 更新chunks
        // ...
        
        // 同步边界
        // ...
        
        // ❌ 每个pass后都调用渲染!
        if (pass_callback_) {
            pass_callback_(pass);
        }
    }
}
```

**问题原因**:
1. 每个pass后都调用`pass_callback_`
2. `pass_callback_`会触发渲染
3. 4个pass会渲染4次
4. 每次渲染都会清空屏幕并重新绘制
5. 导致画面闪烁

## 🔧 修复方案

### 修复内容

**位置**: `src/core/World.cpp` - `World::update()`

**修复代码**:
```cpp
void World::update(float dt) {
    // ...
    
    // 棋盘格4-pass更新
    for (int pass = 0; pass < 4; ++pass) {
        // 更新chunks
        // ...
        
        // 同步边界
        // ...
    }
    
    // ✅ 调用pass完成回调(渲染) - 只在所有pass完成后调用一次
    if (pass_callback_) {
        pass_callback_(3);  // 传递最后一个pass
    }
}
```

**修复原理**:
1. 移除每个pass后的渲染调用
2. 只在所有pass完成后调用一次渲染
3. 避免重复渲染,防止闪烁

## 📊 修复效果对比
